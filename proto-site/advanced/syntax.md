# Έμφαση σύνταξης για γλώσσες προγραμματισμού

Περικλείουμε το πρόγραμμα ανάμεσα σε δύο γραμμές με τριπλούς ανάποδους τόνους. Στην πρώτη γραμμή μπαίνει και η γλώσσα π.χ. `javascript`.

Παράδειγμα:

    ```javascript
    var hello = function () {
        // say hello
        alert('Hello world!');
    }
    ```

Αποτέλεσμα:

```javascript
var hello = function () {
    // say hello
    alert('Hello world!');
}
```
